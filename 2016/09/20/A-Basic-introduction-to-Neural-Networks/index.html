An introduction to neural networks and their implementation in NodeJS - Part 1

2016-09-20

Introduction

A Neural Network is an algorithm used in computer science that tries to mimic the operations of an animal brain.
The first model was built in 1943 by Pitts and McCulloch in a scholar paper called “A logical calculus of the ideas immanent in nervous activity“.

By the way, if you are interested in reading the whole article you can retrieve it there

The first working theories of Neural Nets were inspired by observations of animal brains and especially frog brains. The animal brain was easier to understand because it’s structure is simpler (but extremely similar) than the human one.

The basic aim of an artificial neural network is to give computers the ability to solve complete tasks by providing them adaptive algorithms that are working like the human brain ! What an exciting goal for engineers ! Being able to produce machines that can solve tasks the same way as our brain is an unbelivable opportunity.

In this article I will try to give insights about the way simple neural networks are working. My main aim is to demystify those algorithms, because at the end of the day they are quite simple to understand and at the same time very powerful.

The biological structure of the brain

The human brain weights approximately 1.5 kg. Those 1.5 kg organ is composed approximately of 170 billions of cell with 90-100 billions of neurons. Neurons are the most interesting element of the brain, they are composed of the Soma (the cell body), dendrites (approximately 7.000 per neuron) and a single axon that ends with several connectors (the axon terminal).

The dendrites represent the entrance point (input) and the axon is the output of the system. The all body is transmitting constantly raw data to the brain via the nervous system. You smell an odor in the street, your body will automatically transmit this information in an electrical form to your brain. The role of your brain neurons will be to decode this information and to trigger an action or not.

Those neurons are highly interconnected. Each neuron is connected to another one by axons and dendrites. This structure generates an intense network of computation units connected with one another.

The neuron is made of a very simple algorithm

First input data transmitted by nerves are coming over the dendrites. The neuron then weight the intensity of electrical power. If the level is above a threshold the signal is transmitted to the next neuron(s).

This example is not very interesting but we can consider neural networks made of more neurons that can treat more complex data. Just remember that the role of a neuron is to weight the input signal and then decide if this signal should be propagated to the next neurons. Easy no ?

Application of this knowledge to computer science

Let’s take an example of usage : robotic systems maker make a great use of neural networks. Programers want to be able to construct a model that takes raw data from sensors and that should output a decision about the next move that the robot should perform.

Another example of task is image classification. Neural networks can be used to detect what is represented inside an image. Such an algorithm will take as input each pixel of the image converted in an array of integers that represent the color of each pixel, and then the algorithm should give for that specific image a class.

If we take some distance there is always an input that enters into the system and an output. We can also point out that the output and the output are in a very different format. In the later example we have an array of numbers in input and the name of an animal as output. The first example takes raw numeric data and outputs a command that will represent a movement of the robot.

In fact, neural networks are used a lot for classification tasks, whether you want to give a class to an image or to label correctly an input of data.

Anatomy of a neural network

A neural network is primarily composed of :

  • Neurons (we call them perceptrons) represented by circles
  • Links (a simplification of axons and dendrites), that are represented by lines

Those neurons are organized on three levels (we call them layers)

  1. The input layer (that will receive the input data)
  2. The hidden layer (you can have several hidden layers)
  3. The output layer (that will transmit to the programer the output data)

The task of the developer will prepare a dataset of training that will contain for instance 10.000 labelled pictures that represent cats and other animals.
